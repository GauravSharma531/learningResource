# Road Map for Android
---

## Prerequisites
- Basic Knowledge of Java
- Access to an Android device is helpful, but not mandatory

## Workflow
- Installing Android and SDK Tools
- Setting up Virtual Devices
- Learning UI
- Working with Java
- Working with Internet
- Managing Databases
- Publishing Apps
